Include a summary of another page as a block

Params ( indicates required)
from = "filePath"
the path to the included page relative to the content/ folder
Examples
The following code will include the file content/headless/include-example.md 
{{< pi t = "include" from = "headless/include-example.md" }}
The following code will include the file content/blog/monday/index.md 
{{< pi t = "include" from = "blog/monday" }}
Note that using Hugo path overrides may sometimes cause this method to fail. Linking to the markdown file always works
